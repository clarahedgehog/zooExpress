# Zoo Express

This is a multi-page website built for my web development assignment. The goal was to build a clean, organized site using Node.js and Express that uses templates to display data dynamically.

## How The Site Is Organized

- Server - (index.ejs)
- Routes - to different animal groups and animals
- Views (EJS) -templets
  - Partials
  - Sub Pages

### How To Run It

1. Open your terminal in the project folder.

2. Type npm install to get the necessary files.

3. Type node index.js to start the server.

4. Open localhost:3000 in your browser.
